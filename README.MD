# Ray Tracer (C++20)

CPU-based ray tracer implemented in modern C++20, based on
*Ray Tracing in One Weekend* by Peter Shirley.

This project is developed as part of a Master's thesis and
serves both educational and portfolio purposes.

## Current status
- [x] Image output (PPM)
- [ ] Vector math (vec3)
- [ ] Rays and camera
- [ ] Materials and lighting
- [ ] Acceleration structures

More details will be added as the project progresses.
